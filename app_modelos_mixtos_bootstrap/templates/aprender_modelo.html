¿Qué es un Modelo Mixto?

Un modelo mixto combina efectos fijos y aleatorios. Se utiliza cuando los datos tienen estructura jerárquica o repetida, como estudiantes dentro de aulas o mediciones repetidas de un mismo individuo.

📌 Estructura General:

Y = Xβ + Zγ + ε
  • Xβ: efectos fijos (variables independientes comunes)
  • Zγ: efectos aleatorios (variabilidad entre grupos)
  • ε: error

🎯 ¿Cuándo se usa?

  • Cuando hay datos agrupados (por ejemplo: estudiantes en colegios)
  • Cuando hay mediciones repetidas de los mismos sujetos
  • Cuando se quiere modelar tanto la media como la variabilidad entre grupos

📊 Ejemplo aplicado

Estudiamos la relación entre el rendimiento (Y) y horas de estudio (X1) y motivación (X2), considerando como grupo aleatorio la escuela.

Fórmula: Y ~ X1 + X2 con grupo = Escuela

🧪 Diferencias entre modelos:

Modelo Características
Regresión Lineal No considera estructura de grupos
Modelo Mixto Incluye variabilidad entre grupos como efecto aleatorio
